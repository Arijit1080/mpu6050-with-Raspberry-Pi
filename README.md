# mpu6050 with Raspberry Pi

Accessing MPU-6050 on Raspberry Pi using Python to get temperature, accelerometer and gyroscope data.

## Dependencies

```
sudo apt install python3-smbus

pip install mpu6050-raspberrypi
```

## Run 

```
Python3 mpu_6050.py
```
